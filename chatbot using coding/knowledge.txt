1. What is AI Model Training?

Answer:
AI model training is the process of teaching a machine learning model using data. The model learns patterns and relationships from the training data so that it can make predictions or decisions on new data. During training, the algorithm adjusts its parameters to reduce errors and improve performance.

2. What is Model Optimization?

Answer:
Model optimization is the process of improving the performance of a machine learning model. It helps the model make more accurate predictions while reducing errors. Optimization techniques include tuning parameters, selecting better features, and adjusting training methods to achieve the best results.

3. What is Hyperparameter Tuning?

Answer:
Hyperparameter tuning is the process of selecting the best parameters for a machine learning model before training. These parameters control how the model learns. Examples include learning rate, batch size, and number of layers. Proper tuning improves model accuracy and performance.

4. What is Overfitting?

Answer:
Overfitting occurs when a machine learning model learns the training data too well, including noise and unnecessary details. As a result, the model performs very well on training data but poorly on new or unseen data. Overfitting can be reduced using techniques like regularization and cross-validation.

5. What is Underfitting?

Answer:
Underfitting happens when a model is too simple to learn the patterns in the data. The model fails to capture important relationships, leading to poor performance on both training and testing data. Increasing model complexity or improving features can help solve underfitting.

6. What is Learning Rate?

Answer:
The learning rate controls how quickly a machine learning model updates its parameters during training. A high learning rate may cause the model to miss the optimal solution, while a very low learning rate makes training slow. Choosing the correct learning rate helps the model learn efficiently.

7. What is Batch Size?

Answer:
Batch size refers to the number of training samples processed before the model updates its parameters. Smaller batch sizes make learning slower but more accurate, while larger batch sizes speed up training but may reduce accuracy.

8. What is Training Data?

Answer:
Training data is the dataset used to teach a machine learning model. It contains input features and the correct outputs. The model analyzes this data and learns patterns that help it make predictions on new data.

9. What is Model Evaluation?

Answer:
Model evaluation is the process of measuring how well a machine learning model performs. It uses metrics like accuracy, precision, recall, and confusion matrix to analyze the model's predictions and determine if improvements are needed.

10. What is Accuracy in Machine Learning?

Answer:
Accuracy measures how many predictions made by the model are correct. It is calculated by dividing the number of correct predictions by the total number of predictions. Higher accuracy means the model is performing well.

11. What is Precision?

Answer:
Precision measures how many of the predicted positive results are actually correct. It is important in situations where false positive predictions should be minimized, such as medical diagnosis or fraud detection.

12. What is Recall?

Answer:
Recall measures how many actual positive cases were correctly identified by the model. It focuses on reducing false negatives and is important in tasks where missing a positive case can be serious.

13. What is Feature Selection?

Answer:
Feature selection is the process of choosing the most important input variables for training a machine learning model. Removing unnecessary features improves model performance, reduces training time, and prevents overfitting.

14. What is Regularization?

Answer:
Regularization is a technique used to prevent overfitting by adding a penalty to complex models. It encourages the model to keep parameters small and simple, which helps the model generalize better to new data.

15. What is Cross Validation?

Answer:
Cross validation is a technique used to evaluate the performance of a machine learning model. The dataset is divided into multiple parts, and the model is trained and tested several times. This ensures that the model performs well on different data samples.
